## 六、响应式设计

### 窗口尺寸响应
```python
def on_resize(event):
    width = root.winfo_width()
    if width < 600:
        # 移动端布局
        content.pack(side="top")
    elif width < 900:
        # 平板布局
        sidebar.pack(side="left", fill="y")
    else:
        # 桌面布局
        sidebar.pack(side="left", fill="y")
        detail.pack(side="right", fill="y")
```

### 最小最大尺寸
```python
root.minsize(400, 300)  # 最小尺寸
root.maxsize(1920, 1080)  # 最大尺寸
```
